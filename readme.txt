The iPay plugin allows you to add iPay payment method to your site.
After installation, users can safely make payments through any bank card or through the Bank of Georgia online banking.

== Installation ==

1. Upload the `bog-ipay-payment-gateway` folder to the `/wp-content/plugins` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
